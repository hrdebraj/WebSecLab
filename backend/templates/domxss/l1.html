01Confirm the sink

Open DevTools → Sources. Find the inline script on this page. Identify which API is used to insert the hash content into the DOM.
💡 what to look for
The dangerous DOM sinks to know by heart: innerHTML, outerHTML, document.write, eval, setTimeout(str), element.setAttribute('on*', ...), and jQuery's $() when passed user HTML.

02Show that HTML gets rendered

Navigate to /domxss/l1#<b>hi</b> (url-encode or just type it). If you see bold text, the sink is definitely innerHTML-family.

03Try the obvious (and watch it fail)

Now try #<script>alert(1)</script>. It won't fire. This is not a filter — it's a browser behaviour: <script> tags inserted via innerHTML are parsed but never executed.
💡 browser spec quirk
Per HTML spec, script elements inserted via innerHTML do not execute. This is why DOM XSS payloads nearly always use event-handler attributes on auto-loading elements.

04Pick an element that loads immediately

Which HTML elements automatically trigger a load (and therefore onload or onerror) the instant they're inserted into the DOM?
  • Images with a broken src
  • SVG elements
  • Iframes
  • Video / audio with onerror
Choose one and craft a minimal payload that runs alert(1).

05Graduate from PoC to flag capture

The page exposes window.FLAG only after an XSS handler fires. Replace alert(1) with alert(window.FLAG) (or render the flag into the page). When your payload executes, a flag card will appear.
💡 URL encoding
Browsers decode %XX in the URL before exposing it to JS. You can write the payload literally in the hash bar or URL-encode unsafe characters. # itself must not appear inside the payload or it will be truncated.

06Use Burp DOM Invader

If you have Burp's Chromium browser, open DOM Invader. It will highlight the sink and the source (location.hash) and can auto-suggest exploit shapes. Compare with your hand-crafted attack.

07Reflection

How would you fix this while keeping the feature? Consider textContent, DOMPurify, or an allow-list of tags with no event handlers.

function render() {

  document.getElementById('noteOut').innerHTML = raw;   // <- sink
