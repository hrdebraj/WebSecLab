How to use this lab

  1. Start the stack: docker compose up --build
  2. Visit http://localhost:8080
  3. Configure your browser (Firefox recommended) to proxy through Burp on 127.0.0.1:8080. Since the lab is also on port 8080, either change Burp's listener or use localhost:8080 directly with FoxyProxy.
  4. Open the first level of any category from the sidebar.
  5. Read Challenge, then work through Walkthrough step by step. Use Burp Repeater / Intruder / DOM Invader as each level requires.
  6. When stuck, expand the hint accordions — they nudge you without spoiling.
  7. Click Reset Lab Data in the sidebar any time to re-seed.

⚠️ Warning

This application is deliberately insecure. Do not expose it to untrusted networks. Run only inside Docker, on an isolated lab LAN or a single developer host.
